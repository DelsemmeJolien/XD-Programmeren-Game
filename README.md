# XD-Programmeren-Game
Github Pages: https://delsemmejolien.github.io/XD-Programmeren-Game/

--- MAC & CHEESE ---
GAME: 
Breng mac, de muis terug naar zijn familie.
Vermijd sluwe katten en sprokkel kaas onderweg.

BESTURING:
Spring met de spacebar om katten te vermijden en kaas te verzamelen.
Klik op start om te beginnen.
